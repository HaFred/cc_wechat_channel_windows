## 凭据存储结构 (Windows)

```
%USERPROFILE%\.claude\channels\wechat\
├── account.json                 # legacy（default 账号向后兼容）
├── accounts\
│   ├── default.json             # 默认账号
│   ├── work.json                # 工作号
│   └── personal.json            # 个人号
├── media\
│   ├── default\                 # 默认账号图片
│   └── work\                    # 工作号图片
├── sync_buf_default.txt         # 默认账号同步状态
└── sync_buf_work.txt            # 工作号同步状态
```

## 使用方式 (Windows PowerShell)

```powershell
# 登录
bun setup.ts              # 默认账号
bun setup.ts work         # 工作号
bun setup.ts --list       # 查看所有账号

# 启动
claude --dangerously-load-development-channels server:wechat
$env:WECHAT_ACCOUNT='work'; claude --dangerously-load-development-channels server:wechat
```
